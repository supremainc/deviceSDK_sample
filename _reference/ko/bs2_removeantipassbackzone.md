# BS2_RemoveAntiPassbackZone

주어진 안티패스백 구역을 제거합니다.

## 함수

```cpp
#include "BS_API.h"

int BS2_RemoveAntiPassbackZone(void* context, uint32_t deviceId, BS2_ZONE_ID* zoneIds, uint32_t zoneIdCount);
```

## 파라미터

- \[In\] `context` : Context

- \[In\] `deviceId` : 장치 식별자

- \[In\] `zoneIds` : 제거하고자 하는 anti-passback zone id 리스트

- \[In\] `zoneIdCount` : anti-passback zone id 리스트의 개수

## 반환값

성공적으로 수행될 경우 `BS_SDK_SUCCESS`를 반환하고, 에러가 발생할 경우 상응하는 에러 코드를 반환합니다.
