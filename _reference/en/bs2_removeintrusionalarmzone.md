# BS2_RemoveIntrusionAlarmZone

Removes selected Intrusion Alarm zones.

## Declaration

```cpp
#include "BS_API.h"

int BS2_RemoveIntrusionAlarmZone(void* context, uint32_t deviceId, uint32_t* zoneIds, uint32_t zoneIdCount);
```

## Parameter

- \[In\] `context` : Context

- \[In\] `deviceId` : Device ID

- \[In\] `zoneIds` : List of zone IDs to remove

- \[In\] `zoneIdCount` : Number of zone id list

## Return Value

If successfully done, `BS_SDK_SUCCESS` will be returned.

If there is an error, the corresponding error code will be returned.
