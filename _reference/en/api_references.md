# API & References

You can check the details of the API.

- [Return Code](return_code)

  This page provides the information of the return codes.

- [SDK API](sdk_api)

  This API provides functionality for initializing the SDK and managing the memory.

- [Communication API](communication_api)

  This API allows to connect with the device.

- [Device API](device_api)

  This API allows to control the device.

- [Log Management API](log_management_api)

  This API allows to get/write/delete the log data.

- [Access Control API](access_control_api)

  API for access control, which can manage access authority by individual, group, time schedules.

- [Door Control API](door_control_api)

  API that configures whether the device is set as entrance/exit of the door, how the device will control the door, and the APB settings.

- [Wiegand API](wiegand_api)

  API that composes the Wiegand card readers and configures the Wiegand input/output.

- [Server API](server_api)

  API that can enable server matching or controls the device from the BioStar application.

- [SSL API](ssl_api)

  API that configures the SSL communication

- [User Management API](user_management_api)

  This API allows to get/enroll/delete the user information.

- [Configuration API](configuration_api)

  This API allows to get/set the configuration information.

- [Smartcard API](smartcard_api)

  This API allows to scan the smart card.

- [QR code API](qr_code_api)

  API that provides functions to read and write QR code data.

- [Fingerprint API](fingerprint_api)

  This API allows to scan/verify the fingerprint template.

- [Face API](face_api)

  This API allows to scan/verify the face template.

- [Blacklist API](blacklist_api)

  API that adds a card to the blacklist when a card is lost or to limit access.

- [Slave Control API](slave_control_api)

  API that connects the master and slave device by using the RS-485 network.

- [Zone Control API](zone_control_api)

  API that configures the zone, which can control the entry device and door's operations.

- [Useful Articles](useful_articles)

  Useful articles that explain detailed information.

- [Lift Control API](lift_control_api)

  API that configures the OM-120, which can control lifts.

- [USE Exported Control API](use_exported_control_api)

  API that configures USB Exported.
