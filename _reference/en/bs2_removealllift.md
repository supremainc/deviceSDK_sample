# BS2_RemoveAllLift

Removes all doors.

## Declaration

```cpp
#include "BS_API.h"

int BS2_RemoveAllLift(void* context, uint32_t deviceId);
```

## Parameter

- \[In\] `context` : Context

- \[In\] `deviceId` : Device ID

## Return Value

If successfully done, `BS_SDK_SUCCESS` will be returned.

If there is an error, the corresponding error code will be returned.
