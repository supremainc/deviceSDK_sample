# BS2_SearchDevicesCoreStation

Searches CoreStation devices from the current network.

## Declaration

```cpp
#include "BS_API.h"

int BS2_SearchDevicesCoreStation(void* context);
```

## Paramter

- \[In\] `context` : Context

## Return Value

If successfully done, `BS_SDK_SUCCESS` will be returned.

If there is an error, the corresponding error code will be returned.
